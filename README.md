# Disable Copilot in Comments

A VS Code extension that automatically disables GitHub Copilot inline completions when your cursor is inside a comment line or block.

## Features

- Supports line comments (`//`, `#`, `--`, etc.) and block comments (`/* */`, `<!-- -->`, etc.)
- Supports 50+ languages
- Works in remote editing sessions (SSH, WSL, Containers, Tunnels)
- Status bar indicator shows current state
